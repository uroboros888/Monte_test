# Monte Test

Проект содержит экспериментальный конвейер моделирования. Ключевые материалы:

- [Архитектура решения](docs/architecture.md) — цели, ключевые компоненты, конвейер и контракты данных.
- `docs/` — дополнительная документация (будет пополняться по мере развития проекта).

## Быстрый старт
1. Изучите [архитектурный документ](docs/architecture.md), чтобы понять ключевые сущности и потоки данных.
2. Подготовьте описания сценариев в корпоративном репозитории DSL.
3. Запланируйте разработку компонентов согласно разделу «План развития» документа по архитектуре.

## Поддержка
Вопросы и предложения по архитектуре направляйте в командный канал #monte-architecture.
